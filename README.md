## About Me

Hi! I'm Musab, an AI Engineering student at ITU Lahore and a Full-Stack Developer. I enjoy building everything from low-level assembly to full-stack web and mobile applications, with a strong focus on AI systems and automation.

I'm all about clean setups, optimizing edge-case logic, writing high-performance code, and exploring hardware-level systems.

### Highlighted Projects

- **[LegalEase](https://github.com/musabmushtaq/LegalEase)**: AI legal assistant
- **[DashMaps](https://github.com/musabmushtaq/DashMaps)**: High-performance C++ navigation engine
- **[Memory-Maze](https://github.com/musabmushtaq/Memory-Maze)**: Real-time retro maze game in x86 Assembly
- **[DashSync](https://github.com/musabmushtaq/DashSync)**: Cross-platform implementation of an Android Auto/CarPlay
